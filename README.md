# Project Title

Cminusminus

## Getting Started

First u need create two directories

```
mkdir build
```

```
mkdir exec
```

### Installing

To install software u need to be in 'build' folder and use command

```
cmake ..
```


### Compile

To compile software u need to 

```
cd build
```

then

```
make
```

### Running

To run software use command in 'build' folder

```
./RootProject
```

or

```
../exec/RootProject
```
